󱐱 Header Dashboard
Shows TRACE TUI brand, pulsing live dot, mode (󰒓 NORMAL / 󰒓 HUNTER), and real-time connection counts ( Apps / 󱂇 Conns).
󰲚 Process List
All active processes with connection counts and color-coded risk badges: 󰄬 LOW 󰒓 MEDIUM 󰀪 HIGH 󰈸 CRITICAL.

Navigate with ↑↓ and press Enter to inspect.
ℹ Connections Table
Tab 1 shows protocol (TCP/UDP), local address, foreign address, connection state, and geolocation. Selected row highlighted in blue.
◈ Risk Overview
Tab 2 displays a bar chart of risk scores. Processes are ranked from LOW (green) to HIGH (red) based on connection analysis.
◈ Timeline Chart
Tab 3 shows connection activity over time with a line chart. Track spikes and patterns in real-time network behavior.
󰬒 Actions Panel
10 quick-access commands: Pause/Resume R, Kill X, Search G, Export S, Hunter Mode H, Firewall B, Language L.
⚡ Firewall Manager
Press B to enter firewall mode. 3-column layout: connections ➜ blocked IPs ➜ actions. Block/unblock with Space B U.
◎ Live Search
Press / to open the search bar. Filter processes and connections in real-time with match count feedback.
◎ Multi-Language
Press L for the language selector with 8 locales. Status bar shows live analysis feedback, version, and platform info.
◎ Connection Analysis
Press Enter on any connection for a full intel report: trust score gauge, WHOIS data, traceroute hops, and security analysis with color-coded remarks.

Powerful Features

Everything you need for network investigation and threat detection

🔍

Real-Time Monitoring

Monitor network connections in real-time with live updates and intelligent filtering.

🛡️

Hunter Mode

Filter unsigned processes automatically. Focus on potentially malicious connections with one keystroke.

🌍

GeoIP Identification

Identify connection locations with country flags, geographic coordinates, and WHOIS lookups. Powered by ip-api.com.

📊

Deep Investigations

Launch nslookup, ping, traceroute, and WHOIS queries directly. View results with a trust score gauge and world map.

🔥

Risk Assessment

Automatic risk scoring (LOW/MEDIUM/HIGH/CRITICAL) based on domain mismatch, latency anomalies, and geolocation.

🛑

Firewall Integration

Block suspicious IPs directly from the interface with Windows Firewall integration and visual connection management.
