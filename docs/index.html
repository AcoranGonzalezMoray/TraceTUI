Step 01

Navigation Sidebar

Access the 5 main sections: Networks, Trends, Storage, Libraries, and Containers. Highly responsive and collapsible.

Step 02

Dashboard Overview

Get a bird's-eye view of your system. Monitor live analysis indicators, operation modes, and real-time connection counts.

Step 03

Process Intelligence

Deep dive into active processes. TraceTUI automatically calculates threat levels using advanced heuristics.

Step 04

Connection Granularity

Inspect every single packet. View protocol details, local/foreign endpoints, and geographic origins in one table.

Step 05

Risk Visualization

Visual comparison of security scores. Identify outliers and suspicious behavior with clear bar charts.

Step 06

Activity Timeline

Track network spikes and dips. The real-time timeline helps you identify patterns over time.

Step 07

Rapid Actions

Take control immediately. Kill suspicious processes, export data to JSON, or manage your firewall with single keystrokes.

Step 08

Advanced Analytics

Explore trends and peak usage stats. Understand your network footprint with detailed distribution metrics.

Step 09

Storage Explorer

Integrated file browser for quick access to system files, logs, and process-related artifacts.

Step 10

Binary Inspection

Review loaded DLLs and shared libraries. Verify signatures and paths for all running code.

Step 11

Container Monitoring

Native Docker support. Monitor container health, resource usage, and bridge networks directly from your TUI.

Step 12

Active Firewall

Real-time blocking. Select connections and instantly block malicious IPs with the built-in firewall manager.

Step 13

Instant Search

Find what you need, fast. Full-text search across processes, connections, and files with recursive matching.

Step 14

Global Ready

Multiple language support with a clean select dialog. TraceTUI speaks your language.

Step 16

System Trends

Analyze performance metrics over time. Monitor CPU spikes, memory leaks, and connection peaks with real-time sparklines.

Step 17

Storage Deep-Dive

Browse system files and logs. Inspect process-related artifacts and configuration files directly from the TUI.

Step 18

DOCKER Monitoring

Native container support. Check health status, resource allocation, and bridge networks for all local containers.

Powerful Features

Everything you need for network investigation and threat detection

🔍

Real-Time Monitoring

Monitor network connections in real-time with live updates and intelligent filtering.

🛡️

Hunter Mode

Filter unsigned processes automatically. Focus on potentially malicious connections with one keystroke.

🌍

GeoIP Identification

Identify connection locations with country flags, geographic coordinates, and WHOIS lookups. Powered by ip-api.com.

📊

Deep Investigations

Launch nslookup, ping, traceroute, and WHOIS queries directly. View results with a trust score gauge and world map.

🔥

Risk Assessment

Automatic risk scoring (LOW/MEDIUM/HIGH/CRITICAL) based on domain mismatch, latency anomalies, and geolocation.

🛑

Firewall Integration

Block suspicious IPs directly from the interface with Windows Firewall integration and visual connection management.

📈

Trends & Analytics

Track CPU history, connection peaks, and memory usage with real-time sparklines. Visualize risk distribution across all processes.

💾

Storage Explorer

Browse disk usage, inspect process-related files, and navigate the file system with an integrated file browser and properties panel.

📚

Library Inspection

Review loaded DLLs and shared libraries per process. Verify digital signatures and detect unsigned or suspicious binaries.

🐳

Container Monitoring

Native Docker support with health checks, resource usage, and bridge network monitoring directly from the terminal interface.
